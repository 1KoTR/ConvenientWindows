# Convenient Windows 0

Это нулевая (начальная) версия проекта `Convenient Windows`, который специально создается под ОС `Windows 11`.

## **Version 0**
`Convenient Windows` представляет собой набор полезных ссылок и программ для Windows.

### [Uninstall Tool Setup](https://crystalidea.com/downloads/uninstalltool_setup.exe)
Условно бесплатная программа для полного удаления любой программы на компьютере. Может удалить не только выбранное приложение, но и все его следы в системе. Умеет следить за установкой какого-либо приложения, чтобы пользователь мог видеть, что именно изменяет в системе программа, когда она устанавливается.

### [Process Hacker Setup](https://sourceforge.net/projects/processhacker/files/processhacker2/processhacker-2.39-setup.exe/download)
Аналог стандартному диспечеру задач в Windows, но у Process Hacker'а гораздо больше возможностей и удобств.

### [Sublime Text Setup](https://download.sublimetext.com/sublime_text_build_4126_x64_setup.exe)
Замена блокноту в Windows. Обладает приятным настраиваемым дизайном и огромным функционалом.

### [Nord Pass Setup](https://downloads.npass.app/windows/NordPassSetup.exe)
Один из лучших менеджеров паролей. Намного безопасней и удобней, чем аналоги в браузерах и не только. Есть удобный встроенный генератор случайных паролей.

### Папка "Activation"
Содержит активаторы для `Windows` и `Uninstall Tool`.

### Папка "Customization"
Содержит программы для простого улучшения внешнего вида Wiwdows.

- Папка "MiniBin" содержит программу `MiniBin-6.6.0.0-Setup.exe` и иконки к ней. Программа MiniBin добавляет удобную маленькую корзину в левой части панели задач.

- Программа `translucenttb-2020-2-version-4.exe` добавляет возможность изменять прозрачность панели задач. Можно изменить ее до полной прозрачности. 

### Папка "Tweaks"
Содержит программы для небольшого исправления недостатков Windows.

- Папка "ClassicContextMenu" содержит программу для возвращения в Widnwos 11 контекстного меню (нажатие на ПКМ) из Windows 10. В этой же папке есть программа для отмены этого действия.

- Папка "DragAndDrop" сожержит программу для добавление драг-н-дропа для панели задач в Windows 11. В начале запустите `VC_redist.x64.exe` из этой же папки.

- Папка "RestartExplorer" содержит программу для добавления в контекстное меню возможности перезапустить проводник. В этой же папке есть программа для отмены этого действия.
